Learning management system and online course builder for WordPress.

== Description ==

= e-Learning for WordPress =
Post your courses online with Cursos Wob in a very easy and simple way.
Create courses with individual lessons with videos and pdf.
Sell your courses with Woocommerce.

== Installation ==

= To Install =

* Download the Cursos Wob plugin file
* Unzip the file into a folder on your hard drive
* Upload the /cursoswob/ folder to the /wp-content/plugins/ folder on your site
* Visit your Dashboard -> Plugins and Activate it there.
